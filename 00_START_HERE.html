DMS Implementing Agent

This is the local starter page for the DMS Implementing Agent.

Open the App

Go to the app folder and open index.html in Google Chrome.

The Four Agents

  1. Proposal Preparation Agent
  2. Project Governance Agent
  3. Document Review Agent
  4. Workflow Agent

Important Folders

  • knowledge_base - JSON brain files
  • prompts - copy-ready prompts
  • templates - document templates
  • source_documents - TORs, proposals, policies, frameworks and manuals
  • outputs - generated proposal, review, governance and workflow documents
  • app - local HTML dashboard

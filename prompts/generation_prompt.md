# Generation Prompt

Use this prompt when generating selected Markdown outputs.

```text
Current task:
Generate only the selected Markdown output or outputs.

Inputs to confirm:
- Selected agent: To be confirmed
- Selected output type: To be confirmed
- Source material supplied: To be confirmed
- Required facts supplied: To be confirmed
- Missing facts: To be confirmed

Rules:
- Use the approved Markdown template path when available.
- Generate selected outputs only.
- Do not generate unselected outputs.
- Do not invent missing facts.
- Mark missing facts as To be confirmed.
- Keep the output clear, structured, and beginner-friendly.
```

# System Rules Prompt

Use these rules at the start of DMS Implementing Agent sessions.

```text
You are supporting the DMS Implementing Agent repository.

Active version:
- V1.x Markdown-first static prompt-control system.

Mandatory rules:
- Use Markdown-first output structures.
- Generate selected outputs only.
- Do not generate unselected outputs.
- Mark missing information as To be confirmed.
- Do not assume missing facts.
- Do not add backend code.
- Do not add Python.
- Do not add server routes.
- Do not add DOCX generation.
- Do not edit Word templates.
- Do not add a database or storage layer.
- Keep everything beginner-friendly and maintainable for non-programmers.
```

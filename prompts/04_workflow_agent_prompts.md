\# Workflow Agent Prompts



\## Prompt 1: Design InTouch Workflow



Use 04\_DMS\_WORKFLOW\_AGENT\_CORE. Design a DMS InTouch workflow for the following process:



\[INSERT PROCESS NAME]



Include:

\- purpose

\- trigger

\- main roles

\- special fields

\- workflow stages

\- workflow rules

\- notifications

\- escalations

\- statuses

\- dashboards

\- audit trail requirements

\- InTouch configuration blueprint



\## Prompt 2: Convert SOP to Workflow



Use 04\_DMS\_WORKFLOW\_AGENT\_CORE. Convert the uploaded SOP or business process into an InTouch workflow design. Produce a workflow design document, field register, role matrix, rules register, notification matrix and test scripts.
